This project analyzes retail inventory data using Python and data visualization libraries.  
The analysis helps understand sales trends, stock levels, and product performance.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Features
- Data Cleaning
- Sales Analysis
- Inventory Insights
- Data Visualization
- Correlation Analysis
